Task-Manager - программа для записи и контроля нужных задач. В нём можно добавлять, отслеживать состояние и удалять задачи, так же просмотреть уже выполненные или активные задачи.

Команды запуска: [
npm install
npm run dev
]

По желанию: [
npm run build
npm run lint
]
